## 📌 Project Overview
This project serves as my primary professional hub. It was built with a **Mobile-First** approach, focusing on fluid typography, high-speed performance, and a clean "Light/Orange Gradient" aesthetic. It highlights my ability to create custom UI components without relying solely on heavy third-party libraries.

## 🛠️ Technical Features
- **Fluid Responsive Typography:** Implemented using CSS `clamp()` for seamless scaling across all device sizes.
- **Custom Navigation:** A unique fullscreen mobile menu built with Vanilla JS and Bootstrap Collapse events.
- **Performance Optimized:** Utilizing `.webp` image formats and lazy-loading to ensure a 90+ Google Lighthouse score.
- **Form Integration:** A fully functional contact system using **Web3Forms** with asynchronous `fetch` requests and automated bot-protection.
- **Scroll-Spy & AOS:** Integrated **Animate On Scroll (AOS)** and custom scroll-padding for a smooth user experience.

## 🧰 Tech Stack
- **Structure:** HTML5 (Semantic Architecture)
- **Styling:** CSS3 (Flexbox/Grid, Custom Variables)
- **Framework:** Bootstrap 5.3.3
- **Logic:** Vanilla JavaScript (ES6+)
- **Icons:** Font Awesome 6 (Brands & Solid)
